# LibraryProject

<!-- ABOUT THE PROJECT -->
## О проекте

Реализация веб-приложения для библиотеки. Библиотекари имеют возможность регистрировать читателей, выдавать им книги и освобождать книги (после того, как читатель возвращает книгу обратно в библиотеку).
